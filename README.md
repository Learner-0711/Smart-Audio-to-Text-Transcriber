# Smart-Audio-to-Text-Transcriber

A modern desktop application built with Tkinter that transforms speech into text using the Wav2Vec2 model. Easily upload .wav files and get accurate transcriptions in seconds.

⚙️ Features

📁 Upload .wav audio files for quick transcription

✍️ Convert speech to text with a simple button click

💡 User-friendly and modern interface built with Tkinter

✅ Confirmation message on successful file upload

🔊 Try it out with sample audio files:
  https://www.voiptroubleshooter.com/open_speech/american.html

🚀 Getting Started

```bash
pip install -r requirements.txt
python main.py
